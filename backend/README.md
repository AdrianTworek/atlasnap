# Atlasnap

Travel memory storage and organization SaaS.

## Development

### Prerequisites
- Docker & Docker Compose
- uv (optional, for local development)


# Build and start
```bash
docker compose up --build
```

# Run in background
```bash
docker compose up -d
```

# View logs
```bash
docker compose logs -f api
```

# Stop
```bash
docker compose down
```
